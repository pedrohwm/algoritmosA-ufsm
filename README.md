# Algoritmos e Programação A | UFSM

Este repositório contém as resoluções das atividades e desafios propostos durante a disciplina de **Algoritmos e Programação A**, no primeiro semestre do curso de **Redes de Computadores** na **Universidade Federal de Santa Maria**.

## Sobre o Repositório

O objetivo deste projeto é documentar meu aprendizado fundamental em programação, partindo do zero até a construção de lógicas estruturadas utilizando a linguagem **C**. Aqui estão registrados os exercícios que serviram de base para minha trajetória técnica.

## Tópicos Aprendidos

Durante a disciplina, explorei os conceitos essenciais da computação, incluindo:

- **Introdução à Programação:** Lógica de processamento e instruções.
- **Variáveis e Tipos de Dados:** Manipulação de inteiros, reais e caracteres.
- **Estruturas Condicionais:** Tomada de decisão com `if`, `else` e `switch`.
- **Estruturas de Repetição:** Automação com `for`, `while` e `do-while`.
- **Vetores e Matrizes:** Organização de dados em estruturas lineares e tabelas.

## Tecnologias Utilizadas

- **Linguagem C:** Base para lógica e manipulação de memória.
- **CodeBlocks:** IDE principal para desenvolvimento e depuração.
- **GCC:** Compilador utilizado para executar os programas.
- **VSCode:** Ambiente para organização e versionamento.

## Organização

As atividades estão organizadas por aulas, contendo o código-fonte e o enunciado em PDF:

- **/Aula 1**: Operadores e Lógica Sequencial.
- **/Aula 2**: Estruturas de Decisão e Lógica Condicional.
- **/Aula 3**: Operadores lógicos.
- **/Aula 4**: Estruturas de Decisão Avançadas e Operadores.
- **/Aula 5**: Laços de Repetição e Teoria dos Números.
- **/Aula 6**: Lógica de Operadores e Sequências.
